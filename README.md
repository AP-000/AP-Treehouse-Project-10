## Pros / Cons of Using an AI Coding Assistant

### Pros

_What were the benefits of using an AI assistant during this project?_

- The LLM was able to create this project much faster than doing it manually.
- Copilot was also very helpful when it came to debugging the code.

### Cons

_What were the downsides, limitations, or challenges when relying on AI during this project?_

- I didn't really experience any limitations with using Copilot. It didn't rely too much on context and was able to understand my propmpts and needs based on the code available.
- I do feel a bit fraudulant while using Copilot. I of course have a general understanding of what it is doing, but I feel like using an LLM doesn't give me the same understanding of my work when doing it manually. For example, if i've fully written my code and come back to it, i'd have a much better understanding of the program if I had to come back to it in future (if that makes sense).
